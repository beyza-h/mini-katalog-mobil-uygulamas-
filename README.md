# Mini Katalog Mobil Uygulaması
### Bu uygulama Flutter kullanılarak geliştirilen bir e ticaret mini katalog mobil uygulamasıdır;ürünlerin listelenmesini,ürünlerin incelenmesini,sepete eklenebilmesini sağlar.
##Kullanılan Flutter Sürümü: 3.41.7
###Çalıştırma Adımları 
1.Bu depoyu klonlayın.
2.'flutter pub get' komutunu çalıştırın.
3.'flutter run' ile uygulamayı başlatın.
